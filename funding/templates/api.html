API documentation

Requests are made using standard HTTP and responses are returned in JSON format.

  • /api/1/convert/pk-usd
  • /api/1/proposals


GET /api/1/convert/pk-usd

PK to USD conversion. Data is pulled from CoinGecko.com and CoinMarketCap.com

Example: 
curl -vvX GET 'http://funding.pennykoin.com/api/1/convert/pk-usd?amount=1000000'
Response: 
{"usd": 224.02}
GET /api/1/proposals

Proposal listing.

Parameters:
  • status: the proposal status, from 0 to 5. Default is 1.
  • limit: limit results. Default is 20.
  • offset: offset results. Default is 0.
Example: 
curl -vvX GET 'http://funding.pennykoin.com/api/1/proposals?offset=0&limit=1&status=2'
